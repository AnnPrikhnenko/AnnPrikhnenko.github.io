Программирование

Для 5-6 класса

Рассчитан на изучения Python с нуля. Занимаемся один раз в неделю по полтора часа. В спокойном темпе изучаем все основы Python.

Для 7-8 класса

Рассчитан на изучения Python с нуля. Занимаемся два в неделю по полтора часа. За год ребята научаться программировать на Python и поймут, хотят ли заниматься этим в дальнейшем. Эти занятия подойдут для ребят, которые собираются сдавать огэ и егэ по информатике.

Для продолжающих

Подходит для ребят, которые прошли один из первых курсов или ранее уже занимались Python и хорошо владеют основными его конструкциями.

Математика

Углубленная математика

Подходит для ребят, которые собираются поступать в ШГН, а также для тех, кому кажется математика в школе слишком легкой.
Набираются группы: 4, 6, 7 класса.

Школьная математика

Если школьная программа плохо получается, то наши курсы помогут подтянуть знания по ней.
Набираются группы: 3, 4, 5, 6, 7 класса.
